## 项目描述
### 首先开机启动,有两分钟时间做网络准备，其次每1分钟去ping两次网络，然后根据以下情况操作
**异常情况**
* 如果未能正常访问 自动重启系统
* 网络为其他网络 不做任何操作

**以太网正常**
* 网络正常时 不做任何改变
